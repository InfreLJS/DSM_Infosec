# DSM Infosec Homepage

2017년 3월부터 진행중인 대덕소프트웨어마이스터고등학교 정보보안과의 홈페이지를 제작하는 1인 프로젝트이다.

HTML, CSS, Javascript, PHP, MySQL 등을 이용하여 구현하였다.

기능은 다음과 같다.
- 로그인, 로그아웃, 회원가입, 회원탈퇴
- 자유게시판 글 / 댓글 쓰기, 수정, 삭제
- 개인 레주메 관리
- 학급 공통 과제 관리
- 학급비 관리

현재 Version 3.0 까지 구현이 완료되었다.
3.0 버전은 Ajax 기술을 도입하여 모든 기능이 index.php에서 작동한다.
